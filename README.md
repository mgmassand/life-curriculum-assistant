# Life Curriculum Assistant

A parenting curriculum assistant to help raise children from 0-18 years.

## Features

- Age-appropriate developmental milestones and activities
- Personalized curriculum tracking for each child
- AI-powered parenting advice and suggestions
- Progress tracking and journaling

## Setup

1. Copy `.env.example` to `.env` and configure your settings
2. Run `docker-compose up -d` to start the application
3. Open http://localhost:8000 in your browser
